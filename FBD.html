Fundamentos e Projeto de Banco de Dados

Carga Horária

72h

Ementa

Apresentar sistemas de gerência de banco de dados do ponto de vista de seu uso. Discutir as técnicas para modelar dados e projetar banco de dados.

Competencias

1) Sistemas de gerência de banco de dados: A) conceitos básicos, tipos de abordagens de BD; B) abordagem relacional: modelo de dados e restrições de integridade, álgebra e cálculo relacional; SQL, normalização e dependências funcionais. C) outros aspectos: concorrência, transações, reconstrução.

2) Modelagem e pro-jeto de banco de dados: A) modelagem entidade-relacionamento; B) transformações entre modelos de BD; C) normalização.

Habilidades

Criar e modelar projetos de entidade-relacionamento, como criar e modelar banco de dados reladionais.
